# AZ-900 – Module 1: Cloud Concepts

## ☁️ What is Cloud Computing
Cloud computing is the delivery of computing services over the internet.

Instead of:
- buying physical servers
- managing datacenters

You use on-demand resources and pay as you go.

---

## 💰 Consumption-based model
You only pay for what you use.

Benefits:
- No upfront costs
- Scale up and down
- Cost efficient

---

## 🌍 Types of Cloud

### Public Cloud
Owned by cloud provider (Microsoft Azure)

### Private Cloud
Used by one organization

### Hybrid Cloud
Combination of public + private

---

## 🤝 Shared Responsibility Model

### Cloud provider is responsible for:
- Physical datacenter
- Network
- Hardware

### Customer is responsible for:
- Data
- Access management
- Applications
