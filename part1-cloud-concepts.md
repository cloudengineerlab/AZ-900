# AZ-900 Part 1 – Cloud Concepts

This document summarizes the core concepts of cloud computing covered in Part 1 of the Microsoft Azure AZ-900 learning path.

---

# What is Cloud Computing

Cloud computing is the delivery of computing services over the internet instead of owning and maintaining physical servers.

Organizations can rent computing resources from cloud providers and scale them when needed.

Examples of cloud resources:

- Virtual machines
- Storage
- Databases
- Networking
- Applications

Major cloud providers include:

- Microsoft Azure
- Amazon Web Services (AWS)
- Google Cloud Platform (GCP)

---

# Benefits of Cloud Computing

Cloud computing provides several advantages compared to traditional on-premises infrastructure.

| Benefit | Explanation |
|------|------|
| High Availability | Systems remain operational even if hardware fails because resources are distributed across datacenters. |
| Scalability | Resources can be increased or decreased based on demand. |
| Elasticity | Resources automatically scale up or down depending on workload. |
| Reliability | Data is replicated across multiple datacenters, reducing the risk of data loss. |
| Predictable Costs | Cloud uses a pay-as-you-go model, meaning you only pay for the resources you consume. |
| Global Reach | Applications can be deployed worldwide using multiple regions. |

---

# Cloud Service Models

Cloud services are divided into three main models.

| Model | What the customer manages | What the cloud provider manages | Example |
|------|------|------|------|
| IaaS (Infrastructure as a Service) | Operating system, applications, data | Servers, storage, networking, virtualization | Azure Virtual Machines |
| PaaS (Platform as a Service) | Applications and data | Infrastructure + operating system + runtime | Azure App Service |
| SaaS (Software as a Service) | Only uses the software | Everything (application, platform, infrastructure) | Microsoft 365 |

### Summary

- **IaaS:** most control, but also most responsibility  
- **PaaS:** focus on application development  
- **SaaS:** ready-to-use software via the internet

---

# Cloud Deployment Models

Organizations can deploy cloud resources in different ways.

| Deployment Model | Description |
|------|------|
| Public Cloud | Infrastructure owned and operated by a cloud provider and shared among multiple customers. |
| Private Cloud | Cloud infrastructure dedicated to a single organization. |
| Hybrid Cloud | Combination of public cloud and private infrastructure. |

Hybrid cloud allows organizations to keep sensitive data on-premises while using cloud resources for applications.

---

# Key Takeaways

- Cloud computing delivers IT resources over the internet.
- Major benefits include scalability, reliability, and cost efficiency.
- Three main service models exist: **IaaS, PaaS, and SaaS**.
- Cloud environments can be **public, private, or hybrid**.

These concepts form the foundation for understanding Microsoft Azure and cloud platforms in general.
